# Agroforestry Species Living Guide

Applied agroforestry species guide connecting species curation, classification and educational design.

## Overview

This repository presents a portfolio-facing view of an ongoing applied guide focused on species selection, classification and didactic translation for agroforestry contexts.

It is intended to communicate the direction of the project, its editorial and technical logic, and the author's experience in structuring species-based educational materials for practical agroforestry use.

## What this project demonstrates

This repository communicates work in species information structuring, agroforestry-oriented classification logic, educational design for technical content, and reproducible organization of materials for future field and learning applications.

## Current stage

The guide is under development as part of a broader authorial project. Public contents may evolve as the portfolio version is refined.

## Visibility and protection note

This public repository is maintained as a strategic showcase. Core proprietary assets, full editorial architecture, and product-sensitive materials are intentionally not fully exposed here.
